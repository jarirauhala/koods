Scrapy Web Crawler

Bot that crawls throught news looking for articles not about Covid-19


What does the program do?

The program is a typical web crawler; it loads data from web sites and saves all that is useful. It starts from an initial list of web sites and collects all useful links from them. It then adds the links to the list of web sites to be visited later.

The crawler's crawling has been limited to news provided by the Finnish broadcasting company Yle, because I asked for their permission, which they granted.

The crawler inspects the contents of the news it has downloaded and looks for keywords indicating mentions of the corona virus. It then discards those with found mentions and saves the rest. The crawling was done in spring 2020.

What was it made for?

The crawler was done as a part of my Bachelor's Thesis.

What does the code do?

Main functionalities of the crawler are in class alspider. The crawler calls for the parse function with a response from the web site. From the data useful information, such as header, author, and URLs, are parsed using xpath.

Parsed data is used to make Item objects, which are then passed to pipelines using yield function. Pipelines check for keywords in the news text and saves the data locally.

With my Batchelor's Thesis work the crawler was run in cloud server Scrapinghub.
