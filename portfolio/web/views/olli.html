Finlandia Project


What does the program do?

The program implements queries to a database and organizes, compares, and graphically displays the results.

Originally, the database was retrieved from https://www.finlandiahiihto.fi/Tulokset/Tulosarkisto by Python Selenium script. This implementation was altered to a static sql file, because the script was not implemented by me.

The database is a result archive of an annual cross-country skiing event calle Finlandia-Hiihto, hence the name of the project.

What was it made for?

The program is a course work for Tampere University's course TIE-20200 Ohjelmistojen suunnittelu (Software Design), which took place in winter/spring 2020.

The objective of the work was to implement a program, which fetches data from the given website and presents results from queries given by the user. The results the program should be able to provide were defined in the work instructions.

    Structure

    The program has been implemented with MVC (Model-View-Controller) model.

    Views have been implemented with QML. MainPage presents different types of queries possible. By pressing a button a QuestionWindow is presented, which collects search parameters from the user. If the query has been succesfull, the results are presented in ResultWindow. If not, an ErrorWindow is displayed.

    ClickHandler class in Controller recieves the search parameters from the View. It further sends them to InputValidator, which checks their validity. If valid, ClickHandler asks the Model to make the query and to send data to View. If not, ClickHandler returns an error statement to the View.

    Model recieves the query parameters from Controller and they are passed forward to DataHandler. It formulates an SQL query using QT's QSqlDatabase class. Queries are made to local datablase.db file.
