Button Game

Simple React.js app with Node.js server


What does the program do?

The program is a simple game with nothing else than one button.

Pressing the button costs one point and by pressing there is a chance to win more points.

The Node server has a counter, which is incremented by every press of a button. When the counter hits some preset value, points are awarded fow whoever pressed the button.

The interesting part besides the rotatig picture of me is that anyone can increment the server-side counter and the player needs to time their presses accordingly.

What was it made for?

Button Game was made as a job application assignment in spring 2020.

It might be obvious that I wasn't picked for the role, as they were looking for someone with a bit more than a couple days experience with React.

This game is not to show my wide range of technical skills, but rather to show my ability to pick up a completely new subject and learn how to use it within days.
